# What's TennisRate
This application receives match results and calculate player's rates.

## How to Use
dockerを用いてコンテナを作成してログインする:
```
docker-compose up -d 
docker-compose exec app bash
pip install -r requirements.txt
```

## アプリ起動
#### コンテナに入って下のコマンドを実行
```
yarn install
yarn start PORT=8000
```
